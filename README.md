# VLog字幕ツール

🎬 **VLOG動画の字幕を簡単に抽出・編集・翻訳できる無料ツール**

## ✨ このツールでできること

📺 **動画に書かれた字幕を自動で読み取り** → ✏️ **簡単に編集** → 🌍 **多言語に翻訳** → 📁 **YouTubeで使える字幕ファイルを出力**

### 🚀 3ステップで字幕作成
1. **動画を開く** - VLOG動画ファイルを選択
2. **自動抽出** - AIが字幕を読み取り（数分で完了）
3. **編集・出力** - 内容を確認して字幕ファイルを保存

### 💡 こんな方におすすめ
- YouTube VLOG投稿者（多言語字幕で視聴者を増やしたい）
- 音声なし動画に字幕を追加したい方
- 手動での字幕作成作業を効率化したい方
- プログラミング知識は不要！直感的に操作できます

## 📖 基本的な使い方

### ステップ1: 動画を開く
1. ツールを起動
2. 「動画を開く」ボタンをクリック
3. 字幕が映っているVLOG動画を選択

### ステップ2: 字幕を自動抽出
1. 「自動抽出」ボタンをクリック
2. AIが動画から字幕を読み取り（数分待機）
3. 抽出された字幕が右側のテーブルに表示

### ステップ3: 内容を確認・編集
1. テーブルで字幕の内容を確認
2. 間違いがあれば直接クリックして修正
3. 「QCチェック」で問題がないか確認

### ステップ4: 字幕ファイルを出力
1. 「SRT出力」ボタンをクリック
2. YouTubeで使える字幕ファイル（.srt）が保存されます

## 🌍 多言語翻訳機能

### 翻訳の方法を選べます

#### 🤖 自動翻訳（AI翻訳）
- **Google翻訳**: Googleのクラウド翻訳サービスを利用
- **DeepL**: 高品質な翻訳で有名なDeepL APIを利用
- 設定画面でAPIキーを入力すれば、ボタン1つで翻訳完了

#### 📊 Excel連携翻訳（手動・外部サービス利用）
1. 「CSV出力」で翻訳用ファイルをエクスポート
2. ExcelやGoogle スプレッドシートで開いて翻訳
3. 「CSV読み込み」で翻訳結果を取り込み

> **💡 おすすめ**: 少量なら自動翻訳、大量・高品質重視なら外部翻訳がおすすめです

### 対応言語
英語、中国語（簡体字・繁体字）、韓国語、スペイン語、フランス語、ドイツ語、ポルトガル語、アラビア語など、多数の言語に対応

## 🔍 品質チェック機能

字幕の品質を自動でチェックし、問題があれば教えてくれます。

### チェック内容
- **長すぎる字幕**: 読みきれない長い文章を警告
- **表示時間**: 短すぎる・長すぎる表示時間を指摘
- **重複**: 同じ字幕が複数回表示されていないかチェック
- **空白**: 文字が入力されていない字幕を検出

「QCチェック」ボタンを押すと、自動で問題箇所を教えてくれるので、修正してからファイル出力しましょう。

## ❓ よくある質問・トラブルシューティング

### 🚀 起動・インストール関連

**Q: ツールが起動しない**
- Windows: ウイルス対策ソフトが実行をブロックしている可能性があります
- 実行ファイルを「信頼できるファイル」として設定してください

**Q: インストールができない**
- Pythonバージョンが古い可能性があります（Python 3.8以降が必要）
- `python --version` でバージョンを確認してください

### 📖 字幕抽出関連

**Q: 字幕がうまく抽出されない**
- 動画の字幕が鮮明に見えるか確認してください
- 字幕の色（白文字・黒縁）以外は認識精度が下がる場合があります
- 動画解像度が低すぎる場合、認識が困難な場合があります

**Q: 処理が遅い・重い**
- 動画が長時間（30分以上）の場合、処理に時間がかかります
- メモリ不足の場合、動画を分割して処理することをおすすめします

**Q: 抽出された字幕の内容が間違っている**
- 認識精度はAIの性能に依存します
- テーブルで手動修正してください（ダブルクリックで編集可能）

### 🌍 翻訳関連

**Q: 自動翻訳ができない**
- APIキーが正しく設定されているか確認してください
- インターネット接続を確認してください
- API利用制限に達していないか確認してください

**Q: 翻訳の品質が悪い**
- AI翻訳は完璧ではありません
- CSV出力して手動で修正することをおすすめします

### 📁 ファイル出力関連

**Q: SRTファイルをYouTubeにアップロードできない**
- ファイル名に特殊文字が含まれていないか確認してください
- YouTubeが対応している言語コードか確認してください

### 🔧 その他

**Q: 設定が保存されない**
- 設定ファイルの保存フォルダに書き込み権限があるか確認してください

**Q: エラーが発生した**
- エラーメッセージをコピーして、[GitHub Issues](https://github.com/lancelot89/vlog-subs-tool/issues) で報告してください

## 📁 作成されるファイル

### YouTubeで使える字幕ファイル
```
動画名.ja.srt          # 日本語字幕
動画名.en.srt          # 英語字幕
動画名.zh.srt          # 中国語字幕
動画名.ko.srt          # 韓国語字幕
```

これらの`.srt`ファイルを、YouTubeの「字幕機能」にアップロードするだけで多言語字幕が完成します！

### 翻訳用CSV（Excel連携）
```
subs/
├── 動画名_ja_export.csv     # 翻訳元データ
├── 動画名_翻訳手順.md       # 翻訳方法の説明書
```

作業途中の内容を保存して、後から再開できます。
```

```
